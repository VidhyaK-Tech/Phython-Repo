# 🐍 Python Basic Programs

This repository contains simple Python programs designed for beginners to practice fundamental concepts such as input/output operations, arithmetic calculations, and basic logic building.

---

## 📌 Programs Included

### 1. ➕ Sum of Two Numbers

Takes two integers as input and prints their sum.

---

### 2. ➖ Difference of Two Numbers

Accepts two numbers and displays their difference.

---

### 3. ✖️ Product of Two Numbers

Calculates the product of two user-input numbers.

---

### 4. 🔲 Square of a Number

Takes a number and prints its square.

---

### 5. 📐 Area of a Triangle

Calculates the area using the formula:
**Area = (Base × Height) / 2**

---

### 6. ➗ Quotient and Remainder

Divides two numbers and prints:

* Quotient
* Remainder

---

### 7. 🔢 Power Calculation

Raises a number to the power of another number using exponentiation.

---

### 8. 📅 Days to Weeks Conversion

Converts total days into:

* Weeks
* Remaining Days

---

### 9. 📊 Average of Three Numbers

Calculates the average of three input numbers.

---

### 10. 🎓 Marks, Total & Percentage

Takes marks from 3 subjects and calculates:

* Total Marks (out of 300)
* Percentage

---

## 🚀 How to Run

1. Install Python (if not installed)
2. Save the code in a `.py` file
3. Run using:

```bash
python filename.py
```

---

## 🎯 Learning Outcomes

* Understanding user input (`input()`)
* Type casting (`int()`, `float()`)
* Basic arithmetic operations
* Writing clean and readable code
* Building problem-solving skills

---

## 📚 Suitable For

* Beginners in Python
* School/College students
* Anyone starting programming basics

---

## 💡 Note

These programs are simple and intended for learning purposes.

🐍 Python Conditional Programs (If-Else Based)

This repository contains a collection of beginner-friendly Python programs focused on conditional statements (if-else logic). These exercises help build strong fundamentals in decision-making and logical thinking.

📌 Topics Covered
🔢 Number-Based Programs
Check if a number is Positive, Negative, or Zero
Check if a number is Even or Odd
Find the Largest of Two Numbers
Find the Largest of Three Numbers
Check if a number is a Palindrome

👤 Real-Life Condition Programs
Check Voting Eligibility (Age ≥ 18)
Check if a number is Greater than 100
Calculate Grade based on Marks
Calculate Electricity Bill based on Units

🔤 Character-Based Programs
Check if a character is:
Uppercase
Lowercase
Digit
Special Symbol
Check if a character is a Vowel or Consonant

🔺 Geometry & Logic
Check if a Triangle is Valid
Classify Triangle:
Equilateral
Isosceles
Scalene

📅 Utility Programs
Convert Day Number → Day Name
Convert Month Number → Month Name
Calculate Sum of Even Numbers (1 to N)
